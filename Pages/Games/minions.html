Minions and Meatshields

An Action Platform Game for Android and iOS

Minions and Meatshields is a fantasy platform mobile game set in a dystopian world of zombie bunnies. The game features multiple levels, each with new weapons, objectives, and level designs and is available on both iOS and Android.
