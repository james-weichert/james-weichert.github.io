Wisdom Warriors

james-weichert.github.io/wisdom-warriors/

An online educational video game for 1st grade students created as part of a year-long capstone project in 10th grade. The game teaches core 1st grade curriculum in English language arts, mathematics, science, and social studies through gamification, whereby students are encouraged to solve academic problems to advance the plot in the video game. Coded using HTML and CSS with programming provided through JavaScript. Earned global recognition for outstanding capstone project in the IB Community Blog publication.
